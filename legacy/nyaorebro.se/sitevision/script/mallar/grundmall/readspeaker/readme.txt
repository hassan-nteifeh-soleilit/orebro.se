Readspeaker-modulen
==========================
Inlagd på grundmallen för nya Örebro.

Har egen mall som använder metadatat "readspeaker_class" för vilken/vilka ytor som skall läsas upp.
Metadatat är textsträng med ett eller flera classnamn class1[,class2,class3]. Default classnamn för webbplatsen är "pagecontent".
