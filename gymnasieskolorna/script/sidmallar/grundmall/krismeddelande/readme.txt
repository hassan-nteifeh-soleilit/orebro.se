Krisrutan

Hanterar krismeddelanden som publiceras i ett i scriptet utpekat arkiv.
Arkivet ska ha ett metadata med flerval för att kunna styra på vilka sajter meddelandet ska visas.

Allmänt
Namn:Krismeddelande: Sajtval
Beskrivning: Styr vilka sajter meddelandet visas på. Om inget val görs visas meddelandet på alla sajter. Om en sajt förekommer på flera meddelanden så visas den första i arkivordningen.
Metadatatyp: Flerval
Alternativ: Alternativen som läggs till ska matcha värdena i metadatat Sitenamn på respektive sajt.

Behörighet
Rättigheter: Synlig för alla användare oavsett roll

Avancerat
Identifierare: criseMessageSelector
Arv av värde: Förvalt
